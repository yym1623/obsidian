###### HTTP
---
1. 인터넷에서 데이터를 교환하는 프로토콜
2. 클라이언트 <-> 서버 데이터 통신
3. 여기서 발생하는 정상신호, 비정상 신호를 알아야 오류도 대처한다


###### 요청종류
---
1. GET
2. POST
3. PUT
4. DELETE


###### 요청신호
---
1. 정상신호 - 200
2. 비정상신호
	1. 3xx - 클라이언트쪽 추가적인 데이터 필요
	2. 4xx - 클라이언트 오류
	3. 5xx - 서버 오류
3. 그 외 비정상 신호
	1. cors
	2. corb
